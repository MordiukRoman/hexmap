Hex Map Maker

Hex Map Maker is great for tabletop wargamers and RPG players alike. It allows you to create a hex map that you can download as a fully-editable, vector SVG file that you can use in your favorite drawing program (like Adobe Illustrator, Affinity Designer, or Adobe Photoshop) or open in a code editor. Developers can also copy an array of hexes for use in code.

Enter values for rows and columns. I've pre-filled some starter values, but you may have to play with values and your browser window size. Some tips:

  • Odd number rows and columns look best as even numbers leave a "ragged" end.
  • Pointed-top hexes take up more vertical room, so flat top generally work better.
  • If you still want to use pointed-top hexes, entering at least twice the amount of columns than rows reduces scrolling in this window.
